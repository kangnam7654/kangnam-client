Skills & Agents Integration

Claude Code의 skills/agents를 GUI에서 발견하고 실행

동작 원리

1
Discovery — 앱 시작 시 claude --list-skills / claude --list-agents 로 사용 가능한 목록 수집
2
Display — GUI에서 카드/리스트로 보여주고 클릭으로 실행 가능
3
Invoke — 클릭 시 CLI stdin에 /skill-name args 전송, 또는 Agent tool 호출
4
Stream — 실행 결과를 기존 JSON 스트림 파이프라인으로 렌더링 (동일 UnifiedMessage)

UnifiedMessage 확장

Skills/Agents를 위한 추가 메시지 타입
enum UnifiedMessage {
  // ... 기존 variants ...

  // Agent가 spawn됨
  AgentStart { id: String, name: String, description: String },

  // Agent 진행 상황 업데이트
  AgentProgress { id: String, message: String },

  // Agent 완료
  AgentEnd { id: String, result: String },

  // Skill 호출됨
  SkillInvoked { name: String, args: Option<String> },
}
